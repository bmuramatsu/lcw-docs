About Learner Credential Wallet

Learner Credential Wallet is a cross-platform iOS and Android mobile application for storing, sharing and mangaging digital learner credentials.

Install Learner Credential Wallet for your mobile!

The wallet is based on the learner credential wallet specification developed by the Digital Credentials Consortium. The learner credential wallet specification is based on the W3C Verifiable Credentials data model.

Learner Credential Wallet enables individuals to curate and share their digital credentials. Key features include:

  • Receiving digital credentials from standards compliant issuers via link, QR code, or JSON
  • Displaying and manage digital credentials
  • Sharing one or more credentials as public links or JSON files, e.g., for job applications
  • Adding a digital credential to a LinkedIn Profile
  • Backuping and restoring the wallet
  • Keeping credential access safe with strong encryption best practices

Install Learner Credential Wallet

Install Learner Credential Wallet from the Apple App Store or Google Play Store.

Quick Setup

Quick Setup:

After installation, Learner Credential Wallet walks the user through a setup process. Select Quick Setup to continue.

Create Password:

The user must setup a password to secure the wallet. The user will not be able to recover a lost password, they will need to setup the wallet and request their digital credentials again, or restore from a backup. After entering the requested information select Next to continue.

Register Wallet (in Learner Credential Wallet app):

The user can choose to register the wallet to receive credentials from web applications. Select Go to Register Wallet to register the wallet. Select Next/Skip to skip registering the wallet.

Register Wallet (on Learner Credential Wallet website):

If the user selects Go to Register Wallet, they are taken to the Learner Credential Wallet website to complete the registration processs. Select Register to continue with registration or return to the Learner Credential Wallet app to continue app setup.

Allow Wallet (on Learner Credential Wallet website):

If the user selects Register, they are asked if they will allow the "Learner Credential Wallet, lcw.app" website to manage credentials for them. Select Next to continue or Cancel to cancel the registration process. If the user choses Cancel they will need to return to the Learner Credential Wallet app to continue app setup.

Allow Wallet (on authn.io website):

If the user selects Next, they are asked to confirm if they will allow the "Learner Credential Wallet, lcw.app" website to manage credentials for them. Select Allow to continue or Block to prevent the "Learner Credential Wallet, lcw.app" website from managing credentials. After completing this step the user will need to return to the Learner Credential Wallet app to continue app setup.

Creating Wallet:

After proceeding from Register Wallet, the app completes the setup process. Select Take Me To My Wallet to continue.

Home

From the Learner Credential Wallet Home, the user can access the main navigation of the app and view digital credentials that have been added to the wallet.

Main navigation:

  • Add Credential(s): First time users should skip to this section to add a credential(s).
  • Share Credential(s)
  • Settings

Home provides a list of digital credentials in Learner Credential Wallet. The user can select a credential to view, access shortcuts to manage a credential or add a credential.

Home (with a credential):

Once one or more digital credentials have been added to Learner Credential Wallet, the user can view the list of credentials, select a credential to view and access shortcuts to manage a credential from Home.

Shortcut to Share a Credential:

From Home, the user can Share an individual credential by swiping from the left.

Shortcut to Delete a Credential:

From Home, the user can Delete an individual credential by swiping from the right.
